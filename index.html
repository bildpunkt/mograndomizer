Mograndomizer

A random generator for Distance trackmogrify keyword seeds

Trackmogrify generates tracks based on text and specific keywords you pass it. With this generator, you can build seeds that only consist of these special keywords to spice up the random generation a bit. You can just copy and paste these seeds into the game, no need to type them word by word!
